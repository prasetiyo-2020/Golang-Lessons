## Loop for
``` golang
package main
import "fmt"
func main() {
	for i := 1; i <= 5; i++ {
		fmt.Println(i)
	}
}
```

```
$ go run .
> 1
> 2
> 3
> 4
> 5
```

## Loop for while
```golang
package main

import "fmt"

func main() {
	i := 1
	for i <= 100 {
		fmt.Println("Saya sedang belajar Go : ", i)
		i++
	}
}
```
```
$ go run .
> Saya sedang belajar Go :  1
hingga
> Saya sedang belajar Go :  100
```
